\# Automation Exercise

A Selenium-based automation testing framework built using Java, Maven, and TestNG.



\## Tech Stack

\-Java

\-Selenium WebDriver

\-TestNG

\-Maven



\## Project Status
Initial Maven project setup completed. The framework will be extended in subsequent commits with reusable framework components, page objects, utility classes, and automated test scenarios.
